# ML Models

This directory holds serialised model artefacts generated by the training pipeline.

| File | Description |
|---|---|
| `random_forest_bandgap.joblib` | Trained Random Forest regressor |
| `feature_columns.json` | Ordered list of feature names used by the model |
| `model_card.md` | Auto-generated performance summary |

> These files are generated by `make train` and are git-ignored (except this README).
